Colours

Primary background colour

Hexcode : #eff0f3

Class name: .bg-primary (primary background)
.primary-bg-items (primary text on primary background)

Uses: Overall design scheme of the website


Secondary background colour

Hexcode: #fffffe

Class name: .bg-secondary (secondary background)

Uses: card, input fields, sections etc..

Primary text colour

Hexcode : #0d0d0d

Class name : .primary-text (primary text colour)

Uses: h1,h2,h3

Secondary text colour

Hexcode : #2a2a2a

Class name: .secondary-text (secondary text colour)

Uses: subheadings and paragraphs

Accent colour

Hexcode : #6701A7

Class name: .accent (accent colour)

Uses: CTA's and links.
